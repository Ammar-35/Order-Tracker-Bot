# 📦 Package Tracking Chatbot

## 🚀 Overview

This project is a simple chatbot system that allows users to track packages and raise complaints using a tracking ID. It is implemented using both:

* Python (Command Line Interface)
* HTML, CSS, JavaScript (Web Interface)

---

## ⚙️ Setup / Installation

### 🔹 Python (CLI Version)

1. Install Python (version 3.x)
2. Clone the repository:

   ```bash
   git clone https://github.com/your-username/package-tracking-chatbot.git
   cd package-tracking-chatbot
   ```
3. Run the chatbot:

   ```bash
   python app.py
   ```

---

### 🔹 Web Version

1. Open the project folder
2. Run the Python server:

   ```bash
   python app.py
   ```
3. Open browser and go to:

   ```
   http://127.0.0.1:5000
   ```

---

## 🧠 Approach

* Designed a **menu-driven chatbot**
* Used **input validation** (tracking ID: 4 letters + 3 digits)
* Simulated package statuses:

  * Delivered
  * In Transit
  * Lost
* Implemented **error handling**:

  * Invalid tracking ID format
  * Complaint ID mismatch
* Added **complaint system** with confirmation message

---

## ✨ Features

* Track package using tracking ID
* Displays delivery status
* Shows random delivery agent details
* Complaint registration for lost packages
* Error handling and validation
* Exit option

---

## ⚠️ Error Handling

1. Invalid Tracking ID format → prompts user again
2. Complaint ID mismatch → shows error message
